# List Sources

List all knowledge sources in the database.

---

## Instructions

Query the Knowledge MCP to see all available knowledge sources.

### How to Query

Use the `list_sources` MCP tool (no arguments needed).

### What You'll Get

- **Source name**: Book, paper, or case study title
- **Author**: Who wrote it
- **Type**: Book, paper, blog, case study
- **Topics covered**: What areas the source addresses
- **Extraction counts**: How many patterns, decisions, warnings extracted

### Why Check Sources

- Understand the breadth of knowledge available
- See which domains are well-covered
- Identify gaps in coverage
- Verify authority of recommendations

### Current Sources Include

The knowledge base contains extractions from AI/ML engineering literature covering:
- RAG and retrieval systems
- Fine-tuning and model customization
- Prompt engineering
- LLM evaluation
- MLOps and production systems
- Agent architectures

---

**EXECUTE**: Call `list_sources` tool.
